# # IBM HR Analytics Dashboard

## Project Overview
This project analyzes employee attrition and workforce trends using Power BI. The dashboard provides insights into employee demographics, attrition patterns, overtime impact, and departmental performance.

## Tools Used
- Power BI
- Excel
- DAX

## Key Metrics
- Total Employees
- Active Employees
- Attrition Count
- Attrition Rate (%)
- Average Age
- Average Years at Company

## Dashboard Insights
- Attrition by Department
- Attrition by Job Role
- Attrition by Age Group
- Attrition by Gender
- Attrition by Overtime
- Attrition by Years at Company

## Key Findings
- Employees with 1 year of tenure show the highest attrition.
- Certain job roles experience higher attrition than others.
- Overtime has a noticeable impact on employee attrition.
- Most attrition occurs among employees aged 25–35.
